# Corsa Legends Wiki

Independent English guide site for Cogito's Roblox experience Corsa Legends at
`https://www.corsalegends.wiki`.

The project uses Next.js, OpenNext, and Cloudflare. Its public pages cover:

- current codes with checked dates and source-conflict notes;
- cars, purchase decisions, best/fastest-car comparison and Cash planning;
- full tuning, Supra setups, engine swaps, fusions, gearing and grip;
- races, map navigation, current ALPHA update signals and verified Roblox links;
- safe progression without scripts, executors or money glitches.

## Development

```bash
pnpm install
pnpm dev
pnpm lint
pnpm exec tsc --noEmit
pnpm build
```

## Content Rules

- Keep first-party Roblox facts separate from community cross-checks.
- Add checked dates to code, update and tune-sensitive information.
- Do not publish a permanent car ranking without comparable current data.
- Keep Assetto Corsa, FR Legends, Vehicle Legends and similarly named Roblox games separate.
- Do not provide scripts, executors, modified clients or account-risk shortcuts.
